# Ancient Media Player User Guide

## Introduction

Ancient Media Player is a media app that combines music, radio, TV, YouTube, and local video playback in one place. This guide explains what users can do and how to use the major sections of the app.

## Home Screen

The home screen provides shortcut tiles for important sections and actions. Depending on configuration, users may see shortcuts such as:

- History
- Last added
- Most played
- Shuffle
- Choose Folder
- Radio
- TV
- Videos
- YouTube

Users can tap any visible shortcut to jump directly into that feature.

## Music Playback

Users can:

- Play local songs stored on the device.
- Browse music by songs, albums, artists, genres, playlists, and folders.
- View history, last added, and most played content.
- Shuffle songs.
- Choose a folder and play audio files from it.

## Radio

The Radio section allows users to save and stream internet radio stations.

Users can:

- Add a radio station.
- Edit a radio station.
- Delete a radio station.
- Group radio stations into categories.
- Add, edit, and delete radio categories.
- Add thumbnails to stations and categories.

### Adding A Radio Station

To add a radio station:

1. Open the `Radio` section.
2. Tap the add button.
3. Enter the station details.
4. Choose a category if needed.
5. Add an image by link or upload one from the device.
6. Save the station.

### Adding Images

Users can add images in two ways:

- Paste an image URL.
- Upload an image from the device.

When uploading, the app supports:

- Crop
- Resize
- Compression for smaller thumbnail size

## TV

The TV section is used for TV channels and Christian free movie channels.

Users can:

- Watch saved TV channels.
- Add new TV channels.
- Edit existing TV channels.
- Delete TV channels.
- Organize TV channels into categories.
- Add, edit, and delete TV categories.
- Add thumbnails to channels and categories.

## YouTube

The YouTube section helps users save channels they want quick access to.

Users can:

- Add a YouTube channel.
- Edit a saved YouTube channel.
- Delete a saved YouTube channel.
- Open a saved YouTube channel for viewing.
- Add a thumbnail by image link or upload.

## Videos

The Videos section displays video files stored on the device.

Users can:

- Browse local videos.
- View videos in a dedicated video section.
- Tap a video to start playback.

## Home Screen Management

The app includes a home management option so users can control what appears on the home page.

Users can:

- Enable a home shortcut or feature.
- Disable a home shortcut or feature.
- Remove unused shortcuts from the home page.
- Keep only the sections they use most.

## Bottom Navigation And More

If the number of sections is greater than the available bottom navigation slots, the app uses a `More` section so users can still access the remaining features.
